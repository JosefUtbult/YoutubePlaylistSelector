Welcome

The Youtube Playlist Selector is a tool for seamlessly switching between background music playing in Youtube. It's mainly intended for playing background music during roleplaying games, but can be used however you wish.

To setup a list of playlists, go to . In there you add a new playlist using the plus ( ) button. In the empty field that appears you can then give the button a title and paste in either the youtube URL or playlist ID. To remove a playlist, use the trashcan ( ) button next to it. When you have changed either the title or the playlist ID/URL you can apply the changes by pressing .

Importing and Exporting settings

When you have created a list of playlists, you can export this list by pressing the button. These settings are then downloaded to your computer as a JSON file. If you then want to use these settings again, press the button and select your downloaded file.

If you want to setup your playlists manually, you can download a template by pressing the button. You can then edit this template using a file editor and import them to the site. In the left hand side of the settings menu you will get a notification showing whether the import succeeded or failed if something in the file isn't set up correct.

Note that these settings are stored locally in your browser as a cookie. They are never saved by the server.

Finding the playlist URL

You can find the URL of the youtube playlist by going to Youtube and starting a playlist. Then simply copy the whole URL from the searchbar and paste it into the field in settings.

Note that a playlist must be either public or unlisted in order for this application to work.
